OpenSSL License Change Agreement

This is the server for collecting agreements to the OpenSSL license change. We want to move from the current license (which can also be found in the file LICENSE in any download), to the Apache License Version 2.0.

  • If you received an email from us, please visit that link in a browser (for safety, cut and paste it).
  • If you did not receive an email, perhaps it got lost. If so, you can search the database to see which commits we found that contain your email; see below. If you might have used an old address that no longer works, you can see the full list of authors or search on name or email, below.
  • If you did not receive an email, and think you should have done so, please email license@openssl.org and be as specific as possible. Please allow some time for us to get back to you.

We are grateful to all the contributors who have contributed to OpenSSL and look forward to their help and support in this effort.
